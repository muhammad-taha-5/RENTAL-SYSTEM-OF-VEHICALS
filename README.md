## ğŸš— About the Project
This project is a comprehensive **Vehicle Rental Management System** designed to solve the complexities of modern fleet management. I transitioned this project from a single-file script into a **Modular Multi-File Architecture** to demonstrate professional software engineering standards.

### ğŸŒŸ Problems Solved:
* **Membership Lifecycle:** Automated VIP expiry logic using a custom `Date` class.
* **Complex Billing:** Solved manual calculation errors using **Operator Overloading** for combined invoices (Rent + Late Fines + Service Charges).
* **Inventory Control:** Used **Polymorphism** to manage Cars, Bikes, and Trucks under a unified interface while maintaining unique fuel and rent logic.
